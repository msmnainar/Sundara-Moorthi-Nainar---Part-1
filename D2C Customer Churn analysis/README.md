# D2C Customer Churn Analysis – Data Audit & EDA

## Project Overview

This project focuses on understanding customer churn behavior for a Direct-to-Consumer (D2C) personal care brand. The objective is to analyze customer data, identify key churn drivers, and generate actionable business insights for improving retention.

This submission corresponds to **Part 1: Data Audit and Exploratory Data Analysis (EDA)**.

---

## Objectives

- Perform data quality assessment across multiple datasets  
- Create customer-level features from transactional and behavioral data  
- Identify patterns associated with customer churn  
- Generate hypotheses for churn drivers  
- Provide business insights for retention strategies  

---

## Datasets Used

The analysis integrates multiple datasets:

- **customers.csv** → Customer demographics and attributes  
- **orders.csv** → Transaction data (purchase history, spend, returns, ratings)  
- **support_tickets.csv** → Customer service interactions  
- **web_events_snapshot.csv** → Website and app engagement metrics  
- **intervention_history.csv** → Campaign exposure and marketing actions  
- **churn_labels.csv** → Churn target variable  

---

## Important Assumption

All analysis strictly uses data on or before the snapshot date (2025-09-30) to prevent data leakage.
