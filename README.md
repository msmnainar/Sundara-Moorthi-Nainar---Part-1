
# D2C Customer Churn – Data Audit & EDA (Part 1)

## Objective
This repository focuses on auditing raw datasets and performing exploratory data analysis to understand customer churn patterns for a D2C personal-care brand.

## Datasets
The following datasets are used:
- customers.csv
- orders.csv
- support_tickets.csv
- web_events.csv
- churn_labels.csv
- campaigns.csv
